# Miva-Student-Portfolio

A responsive multi-page portfolio and academic management website built as the COS 106 Introduction to Web Technologies Term Project at Miva Open University.

## Live Demo

https://ijagbemi-david.netlify.app/

## GitHub Repository

https://github.com/daviddamilare1/Miva-Student-Portfolio

---

## Project Overview

This project showcases my academic profile, technical skills, projects, and provides an interactive academic planner for managing personal study tasks.

The website demonstrates the practical application of HTML5, CSS3, and JavaScript while following responsive web design principles.

---

## Features

### Home Page

- Professional hero section
- Student photograph
- Welcome message
- Short biography
- Responsive navigation
- Call-to-action buttons

### About Me

- Educational background
- Career aspirations
- Technical skills
- Hobbies and interests

### Projects

- Three featured projects
- Project descriptions
- Screenshots
- Links to project demos

### Academic Planner

Interactive task manager built with JavaScript.

Users can:

- Add tasks
- Mark tasks as completed
- Delete tasks

### Contact

Contact form containing:

- Name
- Email
- Phone Number
- Message

Includes JavaScript validation for:

- Empty fields
- Valid email format
- Numeric phone numbers
